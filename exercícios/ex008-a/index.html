Principais Formatações

1. Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b>. (Não semântico).

Nesta frase, tenho um termo em destaque usando a tag <strong>. (Semântico).

2. Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag <i>. (Não semântico).

Nesta frase, tenho um termo em ênfase usando a tag <em>. (Semântico).

3. Texto Marcado

Podemos criar também um texto marcado usando a tag <mark>

E no outro parágrafo, temos outro texto marcado no final.

4. Texto grande e pequeno

Estamos criado um texto grande (Usando a tag <big>)

E um texto pequeno (Usando a tag <small>)nesse parágrafo.

5. Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

(Usando a tag <del>).

6. Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Usando a tag <ins;>. (Semântica).

Existe também o texto sublinhado com a tag <u> (Não semântica).

7. Texto Sobrescrito

Para inserir coisa do tipo x20+3. (Usando a tag <sup>).

8. Texto Subscrito

Para inserir coisa do tipo H20. (Usando a tag <sub>).
